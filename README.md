## 📖 项目简介

重庆元膳健康科技有限责任公司致力于通过AI技术创新，提供智能健康膳食解决方案。本项目是一个完整的健康膳食评估平台，包含快速评估、深度分析、个人中心和管理后台等核心功能。

### 核心功能

- 🎯 **快速健康评估** - 1分钟快速完成健康评估
- 📊 **深度健康分析** - 完整的中医体质辨识和营养分析
- 👤 **个人健康中心** - 历史记录、数据统计、个性化建议
- 🔧 **管理员后台** - 用户管理、数据统计、食谱管理
- 📱 **响应式设计** - 完美适配PC和移动端

## 🚀 快速开始

### 在线体验

直接访问：https://yuanshan-health-website.vercel.app

### 本地部署

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/yuanshan-health-website.git

# 进入项目目录
cd yuanshan-health-website

# 用浏览器打开首页
start 元膳健康官网首页（升级版）.html
```

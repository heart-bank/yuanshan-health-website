## 🚀 免费上线步骤

### 第一步：准备服务器（免费或低成本）
1. **购买域名** - 约50-80元/年（阿里云、腾讯云等）
2. **购买服务器** - 推荐：
   - 免费方案：Vercel、Netlify（纯静态网站）
   - 低成本方案：腾讯云轻量服务器约50元/年
3. **配置服务器环境**：
   ```bash
   # 安装Nginx
   apt update
   apt install nginx

   # 配置Nginx
   vi /etc/nginx/sites-available/default
   ```

### 第二步：配置HTTPS（完全免费）
1. **安装Certbot**：
   ```bash
   apt install certbot python3-certbot-nginx
   ```
2. **申请证书**：
   ```bash
   certbot --nginx -d yourdomain.com
   ```
3. **自动续期**：
   ```bash
   certbot renew --dry-run
   ```

### 第三步：上传文件
```bash
# 上传所有HTML文件到服务器
scp -r *.html user@server:/var/www/html/

# 设置权限
chown -R www-data:www-data /var/www/html
```

### 第四步：配置域名解析
1. 登录域名管理控制台
2. 添加A记录：
   - 主机记录：@
   - 记录值：服务器IP地址
3. 等待DNS生效（10-60分钟）

### 第五步：ICP备案（免费申请）
1. 准备材料：
   - 营业执照
   - 法人身份证
   - 网站负责人身份证
2. 通过阿里云/腾讯云提交备案
3. 等待审核（7-20个工作日）
4. 公安联网备案（额外1-2周）

### 第六步：测试上线
1. 测试所有页面访问
2. 测试所有功能正常
3. 测试移动端访问
4. 测试HTTPS证书有效

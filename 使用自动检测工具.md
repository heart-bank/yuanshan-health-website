# 网站自动检测工具使用指南

## 📋 工具说明

**文件名**: 自动检测网站.ps1
**功能**: 持续检测网站状态，成功后自动打开网站并停止
**检测间隔**: 30秒
**最大尝试**: 60次（30分钟）

---

## 🚀 使用方法

### 方法1：双击运行（最简单）

1. 找到文件：`自动检测网站.ps1`
2. 右键点击
3. 选择"使用PowerShell运行"
4. 等待检测完成

---

### 方法2：命令行运行

打开PowerShell，运行：

```powershell
cd "d:\hgp\胡高鹏创业思路\重庆元膳健康科技有限责任公司文件"
.\自动检测网站.ps1
```

---

### 方法3：自定义参数

```powershell
# 自定义检测间隔（秒）
.\自动检测网站.ps1 -CheckInterval 60

# 自定义最大尝试次数
.\自动检测网站.ps1 -MaxAttempts 120

# 自定义网站地址
.\自动检测网站.ps1 -WebsiteUrl "https://your-site.vercel.app"

# 组合使用
.\自动检测网站.ps1 -CheckInterval 10 -MaxAttempts 180
```

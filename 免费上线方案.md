**优点**：

- ✅ PostgreSQL数据库

- ✅ 实时订阅
- ✅ 用户认证

**使用方式**：
```javascript
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  'your-url',
  'your-anon-key'
)

// 保存用户数据
const { data, error } = await supabase
  .from('users')
  .insert([{ name: 'John', age: 30 }])
```

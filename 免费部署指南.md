## 方案一：完全免费部署

### 1. Vercel（推荐推荐⭐⭐⭐⭐⭐）

**优势**：
- ✅ 完全免费（个人账户）
- ✅ 全球CDN加速
- ✅ 自动HTTPS
- ✅ Git自动部署
- ✅ 无限带宽
- ✅ 支持自定义域名

**限制**：
- ⚠️ 仅支持静态网站（HTML/CSS/JS）
- ⚠️ 不支持后端代码（Node.js、Python等）
- ⚠️ LocalStorage数据在用户端，不适合生产环境

**部署步骤**：

#### 第一步：准备代码
```bash
# 将所有HTML文件放入一个文件夹
cd d:/hgp/胡高鹏创业思路/重庆元膳健康科技有限责任公司文件
mkdir yuanshan-website
cp *.html yuanshan-website/
```

#### 第二步：推送到GitHub
```bash
# 初始化Git
cd yuanshan-website
git init

# 添加文件
git add .
git commit -m "Initial commit"

# 创建GitHub仓库（在GitHub网站上创建）
# 关联远程仓库
git remote add origin https://github.com/yourusername/yuanshan-website.git

# 推送
git push -u origin main
```

#### 第三步：部署到Vercel
1. 访问 [vercel.com](https://vercel.com)
2. 使用GitHub账号登录（免费）
3. 点击"New Project"
4. 选择GitHub中的yuanshan-website仓库
5. 点击"Deploy"
6. 等待约30秒，部署完成！

#### 第四步：配置自定义域名（可选）
1. 在Vercel项目设置中添加域名
2. Vercel会提供DNS记录
3. 在域名DNS中添加记录

### 2. Netlify（推荐⭐⭐⭐⭐⭐）

**优势**：
- ✅ 完全免费
- ✅ 全球CDN
- ✅ 自动HTTPS
- ✅ 拖拽部署（无需Git）
- ✅ 支持表单处理

**部署步骤**：
1. 访问 [netlify.com](https://netlify.com)
2. 注册账号（使用Email）
3. 拖拽文件夹到Netlify
4. 输入网站名称
5. 部署完成！

### 3. GitHub Pages（推荐⭐⭐⭐⭐）

**优势**：
- ✅ 完全免费
- ✅ GitHub自带
- ✅ 支持Jekyll等静态生成器

**限制**：
- ⚠️ 仅支持GitHub仓库
- ⚠️ URL必须是 `username.github.io`

**部署步骤**：
1. 创建GitHub仓库，命名为 `username.github.io`
2. 上传HTML文件
3. 启用GitHub Pages（Settings → Pages）
4. 访问 `https://username.github.io`

## 详细部署步骤

### 步骤1：购买服务器（可选）

```bash
# 访问腾讯云
https://cloud.tencent.com/

# 选择"轻量应用服务器"
# 配置：
- 镜像：Ubuntu 22.04 LTS
- 配置：2核2GB
- 流量：100GB/月
- 价格：50元/年

# 购买后获取IP地址
# 例如：123.45.67.89
```

### 步骤2：连接服务器

```bash
# 使用SSH连接
ssh root@123.45.67.89

# 输入密码（购买后发送到邮箱）
```

### 步骤3：安装Nginx

```bash
# 更新软件包
apt update

# 安装Nginx
apt install nginx -y

# 启动Nginx
systemctl start nginx

# 设置开机自启
systemctl enable nginx

# 测试Nginx
# 浏览器访问：http://123.45.67.89
# 应该看到"Welcome to nginx!"
```

### 步骤4：上传网站文件

```bash
# 方法一：使用scp（在本地电脑）
scp -r /path/to/html/* root@123.45.67.89:/var/www/html/

# 方法二：使用Git（推荐）
# 在服务器上
cd /var/www/html
git init
git clone https://github.com/yourusername/yuanshan-website.git .

# 方法三：使用SFTP工具
# 使用WinSCP、FileZilla等工具上传
```

### 步骤5：配置Nginx

```bash
# 编辑Nginx配置
vi /etc/nginx/sites-available/default

# 配置内容：
server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;

    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    # 启用gzip压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml;

    # 缓存静态资源
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}

# 测试配置
nginx -t

# 重载Nginx
nginx -s reload

# 设置权限
chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html
```

### 步骤6：测试访问

```bash
# 在浏览器访问
http://123.45.67.89

# 应该看到元膳健康官网首页
```

## ICP备案

### 为什么需要备案？

⚠️ **重要**：
- 根据中国法律规定，所有在中国大陆托管的网站必须完成ICP备案
- 未备案的网站会被防火墙屏蔽
- 备案完全免费，无需费用

### 准备材料

1. **营业执照**
   - 拍照或扫描件
   - 清晰可见公司名称和统一社会信用代码

2. **法人身份证**
   - 正反面照片
   - 或使用人脸识别

3. **网站负责人身份证**
   - 正反面照片
   - 或使用人脸识别

4. **核验照片**
   - 网站负责人手持身份证正面照

5. **网站信息**
   - 网站名称
   - 网站域名
   - 网站内容描述

### 备案流程

#### 第一步：提交备案申请

```bash
# 通过阿里云备案
https://beian.aliyun.com

# 或通过腾讯云备案
https://console.cloud.tencent.com/beian

# 填写网站信息
- 网站名称：元膳健康
- 网站域名：yuanshan.com
- 网站内容：健康膳食咨询服务
```

#### 第二步：上传材料

1. 上传营业执照照片
2. 上传法人身份证照片
3. 上传网站负责人身份证照片
4. 上传核验照片

#### 第三步：等待审核

- **管局审核**：7-20个工作日
- **短信验证**：网站负责人手机号接收验证码
- **备案完成**：收到备案号

#### 第四步：公安联网备案

1. 访问 [全国互联网安全管理服务平台](https://www.beian.gov.cn)
2. 注册账号
3. 备案成功后30天内必须提交
4. 等待审核：1-2周

### 显示备案信息

## 常见问题

### Q1: 为什么要用HTTPS？

**A:**
- ⚠️ 浏览器会提示"不安全"
- ⚠️ 搜索引擎会降低排名
- ⚠️ 无法使用某些API（如地理定位、通知）
- ✅ HTTPS现在是标配，必须配置

### Q2: Let's Encrypt证书多久续期一次？

**A:**
- 有效期：90天
- 自动续期：Certbot会在到期前自动续期
- 无需手动操作

### Q3: ICP备案需要多长时间？

**A:**
- 提交审核后：7-20个工作日
- 公安联网备案：1-2周
- 总计：约1个月

### Q4: 域名解析需要多长时间生效？

**A:**
- 全球DNS更新：10分钟-24小时
- 通常：30分钟内生效
- 可使用 `nslookup` 命令查询

### Q5: 如何知道网站是否可访问？

**A:**
```bash
# 方法一：Ping命令
ping yourdomain.com

# 方法二：在线工具
# https://www.webpagetest.org/

# 方法三：使用curl
curl -I https://yourdomain.com
```

### Q6: LocalStorage数据会丢失吗？

**A:**
- 用户清除浏览器缓存会丢失
- 更换设备/浏览器会丢失
- 隐私模式无法保存

**解决方案**：
- 上线后必须升级为后端数据库
- 或者使用浏览器IndexedDB（容量更大）

### Q7: 如何备份数据？

**A:**
```bash
# 备份网站文件
tar -czf backup-$(date +%Y%m%d).tar.gz /var/www/html/

# 备份到云存储（可选）
# 腾讯云COS、阿里云OSS等

# 定期备份（使用cron）
crontab -e

# 每天凌晨2点备份
0 2 * * * /usr/bin/tar -czf /backup/backup-$(date +\%Y\%m\%d).tar.gz /var/www/html/
```

### Q8: 如何监控网站性能？

**A:**
```bash
# 使用免费工具
# Google PageSpeed Insights
https://pagespeed.web.dev/

# GTmetrix
https://gtmetrix.com/

# WebPageTest
https://www.webpagetest.org/

# 目标：
- 首屏加载时间 < 2秒
- 性能评分 > 90分
- 移动端体验良好
```

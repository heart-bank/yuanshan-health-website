- [ ] 访问首页：https://yuanshan-health-website.vercel.app
- [ ] 检查页面加载速度（应<3秒）
- [ ] 检查浏览器控制台无错误（F12 → Console）
- [ ] 检查网络请求无404错误（F12 → Network）
- [ ] 检查HTTPS证书正常（地址栏显示锁形图标）

- [ ] 点击"首页" - 应返回首页

- [ ] 选择性别：男/女
- [ ] 填写年龄
- [ ] 填写身高（cm）
- [ ] 填写体重（kg）
- [ ] 选择健康目标
- [ ] 选择慢性病史（可选）
- [ ] 选择菜系
- [ ] 选择忌口（可选）

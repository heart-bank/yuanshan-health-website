## 方案1：GitHub Pages（推荐新手）

### 步骤1：准备文件

1. 在项目文件夹中，准备所有HTML文件：
   ```
   元膳健康科技有限责任公司文件/
   ├── 元膳健康官网首页（升级版）.html
   ├── 快速健康评估.html
   ├── AI健康膳食报告系统_pro.html
   ├── admin-dashboard.html
   ├── register.html
   ├── forgot-password.html
   ├── user-center.html
   ├── 隐私政策.html
   ├── 用户协议.html
   └── .nojekyll  # 空文件
   ```

2. 创建 `.nojekyll` 文件（空文件），告诉GitHub不要处理Jekyll

### 步骤2：创建GitHub仓库

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `yuanshan-health`
   - 选择 Public 或 Private
   - 勾选 "Add a README file"
3. 点击 "Create repository"

### 步骤3：上传文件

**方式A：网页上传（适合新手）**

1. 在仓库页面，点击 "Add file" → "Upload files"
2. 拖拽所有HTML文件到上传区域
3. 在底部填写 commit message: "Initial commit"
4. 点击 "Commit changes"

**方式B：Git命令行（推荐）**

```bash
# 初始化Git
cd "d:/hgp/胡高鹏创业思路/重庆元膳健康科技有限责任公司文件"
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 添加远程仓库
git remote add origin https://github.com/your-username/yuanshan-health.git

# 推送到GitHub
git push -u origin main
```

### 步骤4：开启GitHub Pages

1. 进入仓库页面
2. 点击 "Settings" 标签
3. 在左侧菜单找到 "Pages"
4. 在 "Build and deployment" 中：
   - Source: 选择 "Deploy from a branch"
   - Branch: 选择 "main" → save
5. 等待1-2分钟，刷新页面
6. 看到 "Your site is live at: https://your-username.github.io/yuanshan-health/"

### 步骤5：访问网站

打开浏览器访问：
```
https://your-username.github.io/yuanshan-health/
```

**完成！** 🎉

## 方案2：Vercel（推荐专业部署）

### 步骤1：创建GitHub仓库

参考方案1的步骤2和步骤3，将代码上传到GitHub。

### 步骤2：创建Vercel账号

1. 访问 https://vercel.com/signup
2. 使用GitHub账号登录
3. 完成账号设置

### 步骤3：导入项目

1. 登录Vercel后，点击 "Add New" → "Project"
2. 在 "Import Git Repository" 中选择 `yuanshan-health`
3. 点击 "Import"

### 步骤4：配置项目

在配置页面：
- Framework Preset: 选择 "Other"
- Root Directory: 留空（默认）
- Build Command: 留空
- Output Directory: 留空（默认）
- 点击 "Deploy"

### 步骤5：等待部署

Vercel会自动部署，约30-60秒完成。
部署成功后，显示：
```
https://yuanshan-health.vercel.app
```

### 步骤6：绑定自定义域名（可选）

1. 在项目页面，点击 "Settings" → "Domains"
2. 输入你的域名（如 `yuanshan.com`）
3. 按照提示配置DNS记录
4. 等待DNS生效（几分钟到几小时）

**完成！** 🎉

## 方案3：Netlify（最简单）

### 步骤1：准备文件

将所有HTML文件放在一个文件夹中。

### 步骤2：创建Netlify账号

1. 访问 https://app.netlify.com/signup

3. 等待上传完成（约30秒）

### 步骤4：访问网站

Netlify会自动生成一个随机域名，如：
```
https://wonderful-john-123456.netlify.app
```

### 步骤5：更改站点名称（可选）

1. 点击 "Site settings"
2. 点击 "Change site name"
3. 输入你想要的名称，如 `yuanshan-health`
4. 新域名：https://yuanshan-health.netlify.app

**完成！** 🎉

1. 点击 "New Project" → "Deploy from GitHub repo"

## 📊 集成数据库（Supabase）

### 步骤1：创建Supabase账号

1. 访问 https://supabase.com
2. 点击 "Start your project"
3. 使用GitHub账号登录
4. 创建新项目

### 步骤2：创建数据库表

在SQL Editor中执行：

```sql
-- 用户表
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  phone VARCHAR(20) UNIQUE NOT NULL,
  password VARCHAR(100) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 健康评估表
CREATE TABLE health_assessments (
  id SERIAL PRIMARY KEY,
  user_id INTEGER REFERENCES users(id),
  gender VARCHAR(10),
  age INTEGER,
  height DECIMAL(5,1),
  weight DECIMAL(5,1),
  bmi DECIMAL(4,1),
  goal VARCHAR(50),
  diseases TEXT[],
  cuisine VARCHAR(20),
  avoid_foods TEXT[],
  health_score INTEGER,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 步骤3：获取API密钥

1. 在左侧菜单点击 "Settings" → "API"
2. 复制以下信息：
   - Project URL
   - Anon Key (公开密钥)

### 步骤4：在HTML中集成

```html
<!-- 引入Supabase客户端 -->
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>

<script>
  // 初始化Supabase客户端
  const supabaseUrl = 'YOUR_PROJECT_URL';
  const supabaseKey = 'YOUR_ANON_KEY';
  const supabase = supabase.createClient(supabaseUrl, supabaseKey);

  // 保存健康评估数据
  async function saveAssessment(data) {
    const { error } = await supabase
      .from('health_assessments')
      .insert([data]);

    if (error) {
      console.error('保存失败:', error);
      alert('保存失败，请重试');
    } else {
      alert('保存成功！');
    }
  }

  // 查询历史记录
  async function getHistory(userId) {
    const { data, error } = await supabase
      .from('health_assessments')
      .select('*')
      .eq('user_id', userId)
      .order('created_at', { ascending: false });

    if (error) {
      console.error('查询失败:', error);
      return [];
    }

    return data;
  }
</script>
```

## 📈 集成Google Analytics

### 步骤1：创建Google Analytics账号

1. 访问 https://analytics.google.com
2. 登录Google账号
3. 点击 "开始衡量"

### 步骤2：创建媒体资源

1. 填写账号名称：`元膳健康`
2. 填写媒体资源名称：`官网`
3. 点击 "创建"
4. 选择行业和时区
5. 同意服务条款

### 步骤3：获取跟踪代码

1. 在"数据流"页面，点击"网站"
2. 填写网站URL和名称
3. 获取跟踪ID（如 `G-XXXXXXXXXX`）

### 步骤4：添加到HTML

在每个HTML页面的`<head>`标签中添加：

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', 'G-XXXXXXXXXX');
</script>
```

## 🆘 常见问题

### Q1: GitHub Pages部署后访问404？
**A:** 检查：
1. 仓库名是否正确
2. 文件是否上传成功
3. 是否等待了足够时间（1-2分钟）
4. URL是否正确（需要加上仓库名）

### Q2: Vercel部署失败？
**A:** 检查：
1. GitHub仓库是否为公开
2. 文件是否在根目录
3. 是否有语法错误

### Q3: 数据库连接失败？
**A:** 检查：
1. API URL和Key是否正确
2. 表名是否正确
3. 是否启用了RLS（行级安全策略）

### Q4: 如何更改主页文件名？
**A:** 将首页文件改名为 `index.html`，这样访问域名时会自动加载。

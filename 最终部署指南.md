# 元膳健康网站 - 最终部署指南（一键完成）

## 🚀 一键部署（3分钟完成）

### 准备工作

在开始之前，您需要准备：
1. ✅ GitHub账号（您已登录）
2. ✅ GitHub个人访问令牌（需要创建）

### 创建GitHub令牌（30秒）

1. 访问：https://github.com/settings/tokens/new
2. 配置：
   - Note: 元膳健康部署
   - Expiration: 90 days
   - ✅ 勾选 `repo` 权限
3. 点击 "Generate token"
4. **复制令牌**（格式：`ghp_xxxxxxxxxxxxxxxx`）

---

## 📋 执行部署（复制粘贴即可）

在PowerShell中，复制粘贴以下命令：

```powershell

# 替换为您的GitHub令牌

# 运行部署脚本
.\完整部署脚本.ps1 -GitHubUsername "heart-bank" -GitHubToken $GitHubToken
```

**示例**：

## 🎯 最后一步（2分钟）

脚本执行完成后，Vercel会自动打开，您需要：

1. **登录Vercel**
   - 使用您的GitHub账号（heart-bank）
   - 授权Vercel访问GitHub

2. **导入项目**
   - 点击 "Import Git Repository"
   - 选择 `yuanshan-health-website`
   - 点击 "Import"

3. **部署**
   - 保持默认配置
   - 点击 "Deploy" 按钮
   - 等待30-60秒

4. **完成！**
   - 看到 "Congratulations!" 页面

### 常见问题

**Q: 推送失败，提示Authentication failed？**

**Q: Vercel部署失败？**
A: 确认GitHub仓库有代码，检查网络连接，重新部署。

**Q: 网站无法访问？**
A: 检查Vercel部署状态，等待1-2分钟后重试。

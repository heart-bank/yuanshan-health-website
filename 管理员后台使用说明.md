# 元膳健康 - 管理员后台系统说明

## 系统概述

管理员后台系统为元膳健康提供完整的数据管理、分析和导出功能，主要用于市场调研和融资需求展示。

## 功能模块

### 1. 数据概览
- **核心指标卡片**
  - 总用户数：显示系统注册用户总数
  - 完成评估：完成健康评估的用户数量
  - 平均健康分：所有用户的平均健康评分
  - 今日新增：当天新增的用户数量

- **趋势分析图表**
  - 30天用户增长趋势折线图
  - 健康目标分布饼图

### 2. 用户数据管理
- **用户列表展示**
  - 用户ID、性别、年龄、BMI
  - 健康目标、健康评分
  - 创建时间

- **多维度筛选**
  - 按用户ID搜索
  - 按性别筛选
  - 按健康目标筛选

- **用户详情查看**
  - 完整的用户健康信息
  - 评估结果详情
  - 弹窗展示

### 3. 深度数据分析
- **人口统计学分析**
  - 年龄分布柱状图（18-25、26-35、36-45、46-55、55+）
  - 性别比例

- **健康状况分析**
  - BMI分布饼图（偏瘦、正常、超重、肥胖）
  - 慢性病分布（高血压、糖尿病、高血脂、痛风）

- **中医体质分析**
  - 九种体质分布雷达图
  - 主要体质类型占比

- **关键指标展示**
  - 平均年龄
  - 平均BMI
  - 慢性病患病率
  - 评估完成率

### 4. 数据导出中心

#### 4.1 标准导出
- **用户基本信息**
  - 导出字段：用户ID、性别、年龄、身高、体重、BMI、创建时间
  - 用途：市场调研的基础人口数据

- **健康评估数据**
  - 导出字段：用户ID、年龄、BMI、健康目标、慢性病、健康评分、体质类型
  - 用途：融资报告中的健康分析数据

- **统计分析报告**
  - 包含所有核心指标和分布数据
  - 适合直接用于融资PPT和商业计划书

#### 4.2 自定义导出
- **时间范围选择**
  - 开始日期
  - 结束日期

- **数据类型选择**
  - 全部数据
  - 基本信息（仅人口统计学）
  - 健康数据（仅健康相关）
  - 体质数据（仅体质分析）

- **导出格式**
  - Excel (.xlsx)格式
  - 自动生成文件名（包含日期）

## 数据来源

### 1. 模拟数据
- 系统自动生成100条模拟用户数据
- 涵盖不同年龄、性别、健康状况
- 用于演示和测试功能

### 2. 真实数据
- 用户在健康评估系统中提交的数据
- 自动保存到浏览器LocalStorage
- 管理员登录后可查看所有真实用户数据

## 技术特点

### 1. 数据可视化
- 使用Chart.js库
- 支持多种图表类型：
  - 折线图（趋势分析）
  - 饼图/环形图（分布分析）
  - 柱状图（对比分析）
  - 雷达图（多维分析）

### 2. 数据导出
- 使用SheetJS (xlsx)库
- 支持Excel格式导出
- 支持自定义筛选和时间范围

### 3. 响应式设计
- 适配不同屏幕尺寸
- 移动端友好
- 侧边栏可收起

## 市场调研应用

### 1. 目标用户画像
- **年龄分布**：了解核心用户群体年龄段
- **性别比例**：性别偏好分析
- **地域分布**：可扩展添加地域字段

### 2. 健康需求分析
- **健康目标分布**：减脂、增肌、慢性病管理等需求占比
- **慢性病类型**：高血压、糖尿病等疾病患病率
- **BMI分布**：用户体型健康状况

### 3. 产品优化方向
- 根据用户健康目标优化产品功能
- 根据慢性病分布调整食谱推荐
- 根据体质分布开发针对性服务

## 融资需求应用

### 1. 用户增长数据
- 30天用户增长趋势
- 总用户数量
- 日活跃用户（可扩展）

### 2. 用户参与度
- 健康评估完成率
- 用户健康评分分布
- 用户粘性指标（可扩展）

### 3. 市场价值数据
- 平均年龄（显示目标市场）
- 慢性病比例（显示市场痛点）
- 健康评分提升（显示产品效果）

## 使用指南

### 1. 登录后台
1. 访问官网首页
2. 点击右上角用户图标
3. 输入管理员账号（admin / admin123）
4. 自动跳转到管理后台

### 2. 查看数据
1. 默认显示数据概览
2. 点击左侧导航切换页面
3. 使用筛选器过滤数据
4. 点击"查看"查看用户详情

### 3. 导出数据
1. 进入"数据导出"页面
2. 选择导出类型或自定义导出
3. 设置时间范围（可选）
4. 点击导出按钮
5. 自动下载Excel文件

### 4. 数据分析
1. 进入"数据分析"页面
2. 查看各类分布图表
3. 查看关键指标
4. 用于报告和决策

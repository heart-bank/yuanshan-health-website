### 方案三：使用云服务器（适合长期运营）

#### 使用阿里云/腾讯云/华为云

##### 步骤1：购买服务器
1. 注册账号
2. 选择"按量付费"或"免费试用"
3. 配置：
   - CPU：2核（免费版）
   - 内存：2GB（免费版）
   - 带宽：1Mbps（免费版）
   - 系统：Ubuntu 20.04 LTS

##### 步骤2：安装Web服务器
```bash
# 安装Nginx
sudo apt update
sudo apt install nginx -y

# 启动Nginx
sudo systemctl start nginx
sudo systemctl enable nginx

# 配置防火墙
sudo ufw allow 'Nginx Full'
```

##### 步骤3：上传文件
```bash
# 方式1：使用SCP上传
scp -r /本地路径/* root@服务器IP:/var/www/html/

# 方式2：使用Git
cd /var/www/html
git clone 你的仓库地址
```

##### 步骤4：配置Nginx
```bash
# 编辑配置文件
sudo nano /etc/nginx/sites-available/default

# 修改为：
server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}

# 重启Nginx
sudo systemctl restart nginx
```

##### 步骤5：配置HTTPS（免费证书）
```bash
# 安装Certbot
sudo apt install certbot python3-certbot-nginx -y

# 获取免费SSL证书
sudo certbot --nginx -d yourdomain.com

# 自动续期
sudo certbot renew --dry-run
```

## 🔒 HTTPS证书配置（免费）

### Let's Encrypt（推荐）

#### 使用Certbot获取证书

```bash
# 安装Certbot
sudo apt update
sudo apt install certbot -y

# 获取证书（需要先有域名）
sudo certbot certonly --standalone -d yourdomain.com

# 证书位置：
# /etc/letsencrypt/live/yourdomain.com/fullchain.pem
# /etc/letsencrypt/live/yourdomain.com/privkey.pem
```

#### 配置Nginx使用HTTPS

```nginx
server {
    listen 443 ssl;
    server_name yourdomain.com;

    ssl_certificate /etc/letsencrypt/live/yourdomain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/yourdomain.com/privkey.pem;

    root /var/www/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}

# HTTP自动跳转HTTPS
server {
    listen 80;
    server_name yourdomain.com;
    return 301 https://$server_name$request_uri;
}
```

#### 自动续期

```bash
# 设置自动续期
sudo crontab -e

# 添加定时任务（每月1号凌晨3点）
0 3 1 * * certbot renew --quiet
```

## 🆘 常见问题

### Q1: 使用静态托管可以吗？
A: 可以！本项目是纯静态HTML，非常适合GitHub Pages、Vercel、Netlify等免费托管平台。

### Q2: 需要购买云服务器吗？
A: 如果只是静态展示，不需要。如果需要数据库或动态功能，建议使用免费云服务器或云数据库。

### Q3: HTTPS证书多少钱？
A: 使用Let's Encrypt完全免费，每年自动续期，无需费用。

### Q4: 域名多少钱？
A: 免费域名（.tk、.ml等）完全免费。国内域名（.com、.cn）约10-100元/年。

### Q5: 如何选择部署方案？
A: 
- 快速上线：使用Vercel/Netlify
- 长期运营：使用云服务器
- 个人项目：GitHub Pages
- 企业项目：云服务器

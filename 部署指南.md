# 元膳健康官网 - 部署指南

## 一、部署前准备

### 1.1 必需材料
- ✅ 域名（推荐：yuanjian.com 或 yuanshanjk.com）

## 二、免费部署方案（推荐）

### 2.1 免费服务器方案

#### 方案1：GitHub Pages（纯静态站点）
**优点**：
- 完全免费
- 自动HTTPS
- 全球CDN加速
- 无需维护

**限制**：
- 仅支持静态HTML/CSS/JS
- 不支持后端功能
- 需要使用LocalStorage

**步骤**：
```bash
# 1. 创建GitHub仓库
# 2. 上传所有HTML文件
# 3. 进入仓库Settings > Pages
# 4. 选择main分支，保存
# 5. 等待1-2分钟，访问 https://username.github.io/repo-name
```

**适用场景**：
- ✅ 原型测试
- ✅ MVP验证
- ✅ 个人展示
- ❌ 正式商业站点（需要备案）

#### 方案2：Vercel（免费额度）
**优点**：
- 每月100GB流量免费
- 自动HTTPS
- 全球CDN
- 支持自定义域名

**限制**：
- 免费额度有限
- 不支持后端功能（免费版）

**步骤**：
```bash
# 1. 注册Vercel账号
# 2. 安装Vercel CLI
npm i -g vercel

# 3. 在项目目录执行
vercel

# 4. 按照提示完成配置
# 5. 部署成功，获得临时域名
# 6. 添加自定义域名（可选）
```

**适用场景**：
- ✅ 快速部署测试
- ✅ 中小流量站点
- ✅ 全球访问需求
- ❌ 高流量商业站点

#### 方案3：Netlify（免费额度）
**优点**：
- 每月100GB流量免费
- 自动HTTPS
- 表单功能免费
- 支持Serverless函数

**步骤**：
```bash
# 1. 注册Netlify账号
# 2. 拖拽文件夹到Netlify
# 3. 自动部署完成
# 4. 添加自定义域名（可选）
```

**适用场景**：
- ✅ 需要表单功能
- ✅ 需要自定义函数
- ✅ 快速原型

### 2.2 免费数据库方案

#### 方案1：Firebase（免费额度）
**优点**：
- Google提供，稳定可靠
- 实时数据库
- 免费额度：1GB存储，10GB/月流量
- 自带认证系统

**限制**：
- 免费额度有限
- 数据库查询有次数限制

**配置**：
```javascript
import { initializeApp } from 'firebase/app';
import { getDatabase } from 'firebase/database';

const app = initializeApp({
  apiKey: 'YOUR_API_KEY',
  authDomain: 'YOUR_PROJECT.firebaseapp.com',
  databaseURL: 'https://YOUR_PROJECT-default-rtdb.firebaseio.com',
  projectId: 'YOUR_PROJECT',
  storageBucket: 'YOUR_PROJECT.appspot.com'
});

const db = getDatabase(app);
```

#### 方案2：Supabase（免费额度）
**优点**：
- 开源PostgreSQL
- 免费额度：500MB存储，2GB流量/月
- 自带认证系统
- 支持实时订阅

**步骤**：
1. 注册Supabase账号
2. 创建新项目
3. 获取数据库连接信息
4. 配置环境变量

#### 方案3：继续使用LocalStorage（最简单）
**优点**：
- 完全免费
- 无需配置
- 数据存储在用户浏览器

**缺点**：
- 数据不安全
- 无法跨设备同步
- 清除浏览器数据会丢失

**适用场景**：
- ✅ 原型验证
- ✅ 个人测试
- ❌ 商业生产环境

## 三、免费HTTPS证书配置

### 3.1 Let's Encrypt（推荐）

#### 使用Certbot自动配置
```bash
# 1. 安装Certbot
sudo yum install certbot python2-certbot-nginx  # CentOS
sudo apt install certbot python3-certbot-nginx   # Ubuntu

# 2. 获取证书
sudo certbot --nginx -d yuanjian.com -d www.yuanjian.com

# 3. 自动续期
sudo certbot renew --dry-run
```

#### Cloudflare免费SSL（最简单）
1. 注册Cloudflare账号
2. 添加域名到Cloudflare
3. 修改域名DNS到Cloudflare
4. 开启"Flexible SSL"或"Full SSL"
5. 完成

### 3.2 Vercel/Netlify自动HTTPS
使用Vercel或Netlify部署，自动配置HTTPS，无需任何操作。

## 六、Nginx配置示例

### 6.1 基础配置
```nginx
server {
    listen 80;
    server_name yuanjian.com www.yuanjian.com;
    
    root /var/www/html;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # Gzip压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml;
}
```

### 6.2 HTTPS配置
```nginx
server {
    listen 443 ssl http2;
    server_name yuanjian.com www.yuanjian.com;
    
    ssl_certificate /etc/letsencrypt/live/yuanjian.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/yuanjian.com/privkey.pem;
    
    root /var/www/html;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 缓存静态资源
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}

# HTTP自动跳转HTTPS
server {
    listen 80;
    server_name yuanjian.com www.yuanjian.com;
    return 301 https://$server_name$request_uri;
}
```

## 九、备份策略

### 9.1 自动备份脚本
```bash
#!/bin/bash
# backup.sh

DATE=$(date +%Y%m%d)
BACKUP_DIR="/backup/yuanshan"
SOURCE_DIR="/var/www/html"

# 创建备份
tar -czf $BACKUP_DIR/yuanshan_$DATE.tar.gz $SOURCE_DIR

# 删除7天前的备份
find $BACKUP_DIR -name "yuanshan_*.tar.gz" -mtime +7 -delete

echo "Backup completed: $DATE"
```

### 9.2 定时备份
```bash
# 添加到crontab
crontab -e

# 每天凌晨2点备份
0 2 * * * /path/to/backup.sh
```

## 十二、常见问题

### Q1: 需要备案吗？
**A**: 如果使用国内服务器，必须备案。如果使用香港/美国服务器，无需备案。

### Q2: HTTPS证书要钱吗？
**A**: Let's Encrypt完全免费，Vercel/Netlify自动配置免费HTTPS。

### Q3: 需要买服务器吗？
**A**: 如果使用GitHub Pages/Vercel/Netlify，完全免费。如果需要自定义域名，需要买域名（约50元/年）。

### Q4: LocalStorage够用吗？
**A**: 原型阶段够用，正式上线建议使用数据库。Firebase/Supabase都有免费额度。

### Q5: 如何选择部署方案？
**A**: 
- 原型测试：GitHub Pages
- 中小流量：Vercel/Netlify
- 正式商用：国内服务器 + 备案

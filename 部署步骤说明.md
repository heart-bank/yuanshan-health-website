## 📦 步骤3：在Vercel部署（2分钟）

点击这个链接：
```
https://vercel.com/new
```

### 2. 使用GitHub登录

- 点击 "Continue with GitHub"
- 使用您的GitHub账号（heart-bank）登录
- 授权Vercel访问GitHub仓库

### 3. 导入项目

- 点击 "Import Git Repository"
- 在搜索框中输入：`yuanshan-health-website`
- 找到仓库后点击 "Import"

### 4. 配置部署

保持默认设置即可：
- **Project Name**: `yuanshan-health-website`
- **Framework Preset**: Other
- **Build Command**: 留空
- **Output Directory**: `./`
- 点击 "Deploy" 按钮

### 5. 等待部署

- 部署过程约30-60秒
- 看到 "Congratulations!" 页面表示部署成功

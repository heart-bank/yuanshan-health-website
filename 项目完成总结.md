**步骤1：创建GitHub令牌（30秒）**
```
访问：https://github.com/settings/tokens/new
配置：
- Note: 元膳健康部署
- Expiration: 90 days
- ✅ 勾选 repo 权限
复制令牌
```

**步骤2：执行部署命令（1分钟）**
```powershell
cd "d:\hgp\胡高鹏创业思路\重庆元膳健康科技有限责任公司文件"
$GitHubToken = "您的GitHub令牌"
.\完整部署脚本.ps1 -GitHubUsername "heart-bank" -GitHubToken $GitHubToken
```

**步骤3：在Vercel部署（2分钟）**
```
1. 访问：https://vercel.com/new
2. 用GitHub登录
3. 导入仓库：yuanshan-health-website
4. 点击Deploy
5. 等待30-60秒
```

**步骤4：测试网站（5分钟）**
